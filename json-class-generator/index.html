JSON C# Class Generator

JSON C# Class Generator is a tool that analyzes JSON-formatted data, infers the underlying object structure, and generates strongly-typed C# classes for you.
The following types are supported: typed arrays, typed objects, integers, floats, booleans, strings and nullable types. It is possible to specify whether to generate property-based or field-based POCO classes, and their visibility. Class names and array fields are automatically pluralized/singularized, and you can choose whether to leave the original letter case or to use PascalCase.
